Policy per numero di chiamate

Si paga in base al numero di chiamate effettuate.

Questa policy è stata pensata per API con traffico e tempo di utilizzo costanti, ad esempio response con semplici calcoli matematici o messaggi informativi.

Ogni chiamata effettuata, che rispetti la SLA, ridurrà di 1 il valore della chiave API corrispondente.

NC: Numero di chiamate all'API
VA: Valore attuale chiave API
VF: Valore finale chiave API
Formula: VF = VA - (NC * 1)
